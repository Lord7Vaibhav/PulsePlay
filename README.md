🎵 PulsePlay – Spotify Clone
PulsePlay is a Spotify-inspired Android application built with Kotlin and XML layouts. It integrates Spotify’s web player using WebView while offering a sleek custom UI for playback controls.

✨ Features
WebView Integration – Browse Spotify directly inside the app.

Dark Theme UI – Modern design with a stylish look.

Song & Artist Display – Shows current track info with TextView.

Play/Pause Toggle – Interactive button with dynamic icons.

SeekBar Progress – Visual track progress bar.

Toast Notifications – Instant feedback for playback state.

🛠️ Tech Stack
Kotlin – Core app logic.

XML Layouts – UI design.

Android Manifest – Permissions & activity setup
